# Sample Input Files

The application uses SQLite database files as its primary inputs.

## Input files used by the project

- `database/main.db`
  - The source database created by `app.mock_backup.init_source_db()`.
  - Includes `users` and `transactions` tables with sample rows.

- `database/backup/backup_<timestamp>.db`
  - Generated by `app.mock_backup.create_backup()`.
  - Represents a historical backup file that may be restored and verified.

- `database/sandbox/sandbox_database.db`
  - The sandbox database file created by `app.verifier.restore_backup()`.
  - This file is a copy of the selected backup and is used for verification.

## Notes

- The repository `.gitignore` excludes `*.db`, so actual database files are not tracked here.
- To generate sample backups, run the simulator or `app.mock_backup.init_source_db()` and `create_backup()` directly.
