# Sample Data Folder

This directory documents the input files used by the simulator and the expected outputs generated by the backup verification workflow.

## Folder structure

- `inputs/` — describes the backup and sandbox database files the app uses.
- `expected_outputs/` — example JSON and text output produced by verification and AI validation.

## Purpose

These files are intended to help reviewers and developers understand the behavior of the project without requiring actual proprietary `.db` files.

Use this folder to compare actual runtime outputs against the examples below.
